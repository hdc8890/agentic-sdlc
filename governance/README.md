# Governance

This directory defines how the Agentic SDLC platform evolves without fragmenting into incompatible implementations.

## Core governance documents

- [`working-group.md`](working-group.md)
- [`conformance.md`](conformance.md)

## Governance goals

- maintain the semantic model
- review changes to contracts and interfaces
- evaluate compatibility impact
- keep examples and adapters aligned with the normative model
