# Governance

This directory defines how the Agentic SDLC platform evolves and who is responsible for what.

## Core governance documents

- [`working-group.md`](working-group.md) — ownership model and review responsibilities

## Governance goals

- maintain the shared contract model (`contracts/v1/`)
- review changes to schemas and platform entities
- evaluate compatibility impact of contract changes
- keep CLI, examples, and docs aligned with the contracts
